# FinanceDataExplorer

## 📋 *Описание*
Backend-cервис для анализа личных финансов на Python.
Обрабатывает историю операций из Excel, агрегирует доходы и расходы, получает актуальные курсы валют и цены акций через внешние API, рассчитывает выгодные категории кэшбэка и формирует JSON-отчёты.

## 📈 *Для кого этот проект*
Проект может быть полезен:
- для автоматизации анализа личных финансов
- для построения backend-сервисов обработки табличных данных
- как пример архитектуры Python-проекта с разделением бизнес-логики


### 🎯 *Возможности*
- анализ расходов и доходов за выбранный период
- агрегация трат по категориям
- расчёт наиболее выгодных категорий повышенного кэшбэка
- получение курсов валют через внешний API
- получение цен акций через внешний API
- отчёт по средним тратам по дням недели
- сохранение результатов в JSON-файлы

### 💡 *Какую задачу решает проект*
Проект автоматизирует базовый анализ личных финансов без ручного подсчёта в таблицах:
- показывает структуру расходов
- помогает понять, какие категории приносят максимальную выгоду по кэшбэку
- дополняет финансовый отчёт внешними данными по валютам и акциям


### ⚙️ *Установка и запуск*
1. Клонируй репозиторий:
```
git clone git@github.com:Smotrichenko/FinanceDataExplorer.git
cd FinanceDataExplorer
```
2. Установи зависимости через Poetry
```
poetry install
```

3. Подготовка переменных окружения
```
создай .env на основе .env_template.
```

4. Подготовка входных данных
```
помести Excel-файл с операциями в папку data/(по умолчанию, operations.xlsx)
```   
5. Запуск проекта
```
poetry run python main.py
```
После выполнения отчёты сохраняются в папку reports/ в формате JSON.
