# **FinanceDataExplorer**

## **Описание**
Проект помогает анализировать личные финансы: смотреть расходы и доходы, проверять курсы валют, цены акций, считать кэшбэк и строить простой отчёт по дням недели.

## **Установка и запуск**
1. Клонируй репозиторий:
```
git clone git@github.com:Smotrichenko/FinanceDataExplorer.git
```
2. Установи зависимости через Poetry
```
poetry install
```
3. Положить файл с операциями в папку data/ (по умолчанию это operations.xlsx)
   
4. Запуск проекта
```
poetry run python main.py
```
5. Результаты сохраняются в папку reports/ в виде .json файлов.
   
## **Дополнительная информация по проекту**
### **Основные модули:**
*Views.py:*
- events_page — формирует общую картину: расходы, доходы, валюты и акции за выбранный период.

*Services.py:*
- analyze_cashback — анализирует, какие категории были бы самыми выгодными для повышенного кэшбэка.

*Reports.py:*
- report_to_file — декоратор, который сохраняет результат функции в JSON.
= spending_by_weekly — считает средние траты по дням недели за последние 3 месяца.

### **Вспомогательные функции:**
*Utils.py*
- get_date_range — определяет даты для анализа (неделя, месяц, год).

- filter_data — фильтрует таблицу по датам.

- calc_expenses — считает расходы по категориям.

- get_currency_rates — берёт курсы валют через API.

- get_stock_prices — берёт цены акций через API.
